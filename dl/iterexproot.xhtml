About the self-power function

Definition

For any complex number x, this function is defined as

tetrate_2 ⁡ x := exp x 2 ⁡ 1

See iterated exponential notation for more information.

Relations

For any positive integer n, tetrate_2 ⁡ n = n ↑↑ 2
