About the Knoebel-H function

The Knoebel-H function

Definition

For any complex number x, this function is defined as

knoebel_h ⁡ x := lim n → ∞ exp x n ⁡ 1

See iterated exponential notation for more information.

Relations

Equivalent definitions knoebel_h ⁡ x = exp x ∞ ⁡ 1 = x ∞ = x x x . . .

Semantically, and structurally, the equation below holds for all complex number n inside of the Shell-Thron region, but Knuth up-arrow notation is only defined for positive integer n, so it holds for n = 1, knoebel_h ⁡ n = n ↑↑ ∞

Series of knoebel_h ⁡ x + 1

For all real number x ∈ 0 2 the formal power series knoebel_h ⁡ x + 1 = ∑ k=0 ∞ A k ⁢ x k k! = ∑ k=0 ∞ x k k! ⁢ ∑ j=1 k k j j + 1 j - 1

where k j are the (signed) Stirling numbers of the first kind.

In Mathematica, these coefficients are

A[k_] := Sum[StirlingS1[k, j]*Sum[Binomial[j, i]*i^(j - i), {i, 0, j}], {j, 1, k}]
