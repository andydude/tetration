superroot_3

About the super cube root function

Definition

For any complex number x, this function is defined as

f ∂ k j f ∂ k ∘ j

superroot_3 ⁡ x := ~ ↑↑ 3 - 1 ⁡ x

See iterated exponential notation for more information.

Relations

Equivalent definitions superroot_3 ⁡ x = exp x 2 ⁡ 1 = x 2 = x x

For any positive integer n, superroot_3 ⁡ n = n ↑↑ 2
