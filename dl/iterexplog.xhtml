iterexplog

About iterated exponential logarithms

Definition

For any complex number x, this function is defined as

iterexplog ⁡ x y z := slog x ⁡ y - slog x ⁡ z

See iterated exponential notation for more information.

Relations

Equivalent definitions iterexplog ⁡ x y z = w where exp x w ⁡ z = y
