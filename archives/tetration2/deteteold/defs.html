About the Definitions


         The following definitions are based on widely available definitions, including Munafo's definition of the hyper-operators. The only new definition here is the definition of the super-logarithm, which is discussed in more detail in my paper. You can access my paper from the menu above.

Definitions


Definition HY.         Hyper-n Operators         
    x(n)y = "x hyper-n y"
  • x(n)1 = x         
    		•  if n ∈ Z+, n > 1
  • x(n)y = x(n − 1)x(n)(y − 1)         
    		•  if n ∈ Z+, n > 0
  • x(0)y = x ° y         
    		•  "zeration" (the successor function, in most cases)
  • x(1)y = x + y         
    		•  "addition"
  • x(2)y = x y         
    		•  "multiplication"
  • x(3)y = xy = 1xy = ey log(x)         
    		•  "exponentiation"
  • x(4)y = yx = yx1         
    		•  "tetration"
Definition SL.         Super-logarithm         
    slogx(z) = "base x superlog of z"
  • slogx(z) = y         
    		•  if z = yx
  • slogx(z) = slogx(xz) − 1         
    		•  if x > 1, z ≤ 0
  • slogx(z) = slogx(logx(z)) + 1         
    		•  if x > 1, z > 1
  • slogx(z) = −1 + ∑k=1..∞ vk(x) zk / k!         
    		•  if x > 1, 0 < z ≤ 1

    where vk(x) is the k-th element of the
    first column of the inverse matrix of:
    {{ k j / k! − δjk log(x)−j }k=1..∞ }j=0..∞
